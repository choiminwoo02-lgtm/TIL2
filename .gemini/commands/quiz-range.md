# Quiz Range Check Command

## Description
`questions.json` 파일의 문제 중 지정한 시작($1)과 끝($2) 인덱스 범위에 해당하는 문제들을 추출하여 검토합니다.

## Prompt
"현재 프로젝트의 `questions.json` 파일에서 `id` 값이 `$1` 이상이고 `$2` 이하인 모든 문제를 찾아줘.

각 문제에 대해 다음 사항을 검토해서 보여줘:
1. 문제 내용 (ID: X, 카테고리: Y)
2. 모든 선택지의 내용
3. 정답이 올바른지 여부
4. 해설이 적절한지 여부

결과는 아래 형식으로 목록화해서 보여줘:
- [ID: X] 문제 내용 (정답: Y)
"
