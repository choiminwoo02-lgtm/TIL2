# Quiz Add Command

## Description
사용자가 입력한 `$1`(카테고리)과 `$2`(난이도: 쉬움, 보통, 어려움)를 바탕으로 `questions.json`에 추가할 새로운 퀴즈 문제를 생성합니다.

## Prompt
"현재 프로젝트의 `questions.json` 파일 형식을 확인하고, 다음 조건에 맞는 새로운 퀴즈 문제 1개를 생성해 줘.

**조건:**
- **카테고리**: `$1`
- **난이도**: `$2` (쉬움, 보통, 어려움 중 하나)
- **문제 형식**: 4지선다형 (객관식)

**검증 가이드라인:**
1. **정확성**: 제시된 정답(`correct_answer_index`)이 `options` 배열의 내용과 완벽히 일치해야 함.
2. **해설 충실도**: `explanation`은 정답에 대한 이유를 명확하게 설명해야 함.
3. **중복 방지**: 기존 `questions.json`에 이미 존재하는 문제와 겹치지 않아야 함.
4. **ID 자동 할당**: 현재 가장 높은 `id` 값에 1을 더한 값을 `id`로 부여함.

결과는 `questions.json`에 바로 추가할 수 있는 **JSON 객체 형태**로 출력해 줘.
"
