# PRD Step 1: 냉장고 이미지 인식 및 재료 추출

## 1. 목적
사용자가 업로드한 냉장고 내부 사진에서 식재료를 자동으로 식별하여 리스트화합니다.

## 2. 주요 기능
- **이미지 업로드**: 사용자가 카메라로 촬영하거나 갤러리에서 선택한 냉장고 사진 업로드.
- **AI 비전 분석**: `google/gemma-3-27b-it:free` 모델을 사용하여 이미지 내 식재료 식별.
- **재료 리스트 출력**: 인식된 재료들을 텍스트 리스트 형태로 사용자에게 확인 및 수정 가능하게 제공.

## 3. 기술 스택 및 API
- **AI Model**: `google/gemma-3-27b-it:free` (OpenRouter API)
- **Frontend**: React (또는 사용자 선호 프레임워크)
- **Input**: `multipart/form-data` (Image) 또는 Base64 encoded string

## 4. 사용자 흐름 (User Flow)
1. 사용자가 메인 화면의 '냉장고 촬영/업로드' 버튼 클릭.
2. 이미지 선택 및 전송.
3. AI가 이미지를 분석하여 "양파, 달걀, 우유, 버터" 등의 재료 리스트 반환.
4. 사용자가 잘못 인식된 재료를 삭제하거나 빠진 재료를 직접 추가.

## 5. 성공 지표
- 이미지 인식 정확도 (사용자 수정 비율 20% 미만 목표)
- 분석 완료 시간 (5초 이내)
