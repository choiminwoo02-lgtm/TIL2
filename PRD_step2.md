# PRD Step 2: 레시피 생성 및 맞춤형 추천

## 1. 목적
1단계에서 인식된 식재료들을 바탕으로 실질적인 요리 레시피를 생성하여 제안합니다.

## 2. 주요 기능
- **프롬프트 엔진**: 추출된 재료 목록을 기반으로 최적의 요리를 선정하기 위한 프롬프트 작성.
- **레시피 생성**: `qwen/qwen3.6-plus:free` 모델을 통해 요리 이름, 필요한 재료, 단계별 조리법 생성.
- **다양한 옵션 제공**: '간단 요리', '든든한 한 끼', '술안주' 등 테마별 레시피 제안 가능.

## 3. 기술 스택 및 API
- **AI Model**: `qwen/qwen3.6-plus:free` (OpenRouter API)
- **Data Format**: JSON (구조화된 요리 데이터)

## 4. 사용자 흐름 (User Flow)
1. 1단계에서 확정된 재료 리스트를 바탕으로 사용자가 '레시피 추천받기' 버튼 클릭.
2. AI가 재료 조합을 분석하여 최적의 요리법 생성.
3. 요리 제목, 소요 시간, 난이도, 단계별 가이드 출력.
4. 사용자가 '다른 레시피 보기' 클릭 시 새로운 요리 제안.

## 5. 성공 지표
- 레시피 생성 속도 (3초 이내)
- 사용자 만족도 (좋아요/저장 비율)
- 레시피 실현 가능성 (보유 재료 활용도 80% 이상)
