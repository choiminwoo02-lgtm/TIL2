# 상식 퀴즈 게임 PRD (Product Requirements Document)

## 1. 프로젝트 개요
사용자가 다양한 카테고리의 상식 문제를 풀며 지식을 쌓고, 자신의 점수를 기록할 수 있는 인터랙티브 퀴즈 게임입니다.
- **플랫폼**: 웹 브라우저 (Web Based)
- **디자인 테마**: 레트로 & 픽셀 (Retro/Arcade) - 도트 그래픽, 픽셀 폰트, 통통 튀는 색감 활용

## 2. 주요 기능 및 요구사항

### 2.1. 퀴즈 시스템
- **문제 형식**: 4지선다형(객관식)
- **카테고리 구성**: 한국사, 과학, 지리, 일반상식 (각 10문제)
- **총 문제 수**: 40문제

### 2.2. 핵심 게임 메커니즘 (Arcade Style)
- **시간 제한 (Time Limit)**: 문제당 15초 제한. 남은 시간에 비례하여 보너스 점수 지급
- **생명 시스템 (Life/Heart)**: 초기 하트 3개 지급. 오답 시 하트 1개 차감, 모두 소진 시 게임 오버
- **콤보 시스템 (Combo)**: 연속 정답 시 콤보 카운트 증가 및 점수 배율(Multiplier) 적용
- **아이템 시스템 (Items)**: 게임당 각 1회 사용 가능
    - **50:50**: 오답 2개를 무작위로 제거
    - **시간 연장**: 제한 시간 10초 추가
    - **힌트/정답**: 정답에 대한 힌트 제공 또는 즉시 정답 처리

### 2.3. UI/UX 및 피드백
- **비주얼**: 8비트 아케이드 게임 스타일의 UI 요소 (버튼, 배경, 프로그레스 바)
- **화면 구성**:
    - 상단: 하트(Life), 현재 점수, 콤보 표시
    - 중앙: 타이머 게이지, 문제 텍스트, 4개의 답안 버튼
    - 하단: 사용 가능한 아이템 슬롯
- **피드백**: 정답/오답 애니메이션, 콤보 효과음, 게임 오버/클리어 연출
- **진행 상황**: 현재 진행 상황 표시 (예: 1/10 문제)

### 2.4. 점수 및 기록 관리
- **저장 방식**: 웹 브라우저의 로컬 저장소(**LocalStorage**)를 활용하여 개인 최고 기록 관리
- **기능**: 게임 종료 후 최종 점수 산출, 사용자 이름 입력 및 로컬 순위표(Ranking) 업데이트
- **조회**: 메인 화면에서 내 최고 기록 및 과거 기록(Top 10) 조회 가능

## 3. 사용자 흐름 (User Flow)
1. **메인 화면**: '게임 시작' 버튼 및 '내 랭킹 보기' 버튼
2. **카테고리 선택**: 4가지 카테고리 중 하나를 선택하거나 '랜덤 모드' 선택
3. **퀴즈 진행**: 문제 표시 -> 사용자 입력 -> 즉각 피드백(콤보/이펙트) -> 다음 문제 이동
4. **결과 화면**: 최종 점수 및 통계 확인 -> 이름 입력 및 저장
5. **순위표**: 로컬에 기록된 랭킹 출력

### 2.5. 콘텐츠 관리
- **문제 저장**: 모든 퀴즈 데이터는 로컬 프로젝트 내의 `questions.json` 파일에 정적으로 저장되어 관리됨
- **문제 수**: 카테고리별 10문제씩 총 40문제가 고정된 데이터셋으로 제공됨

## 4. 데이터 구조 (예시)
- **Question Set (questions.json)**:
  ```json
  [
    {
      "id": 1,
      "category": "history",
      "question_text": "조선 시대 제1대 왕은 누구입니까?",
      "options": ["태조", "정종", "태종", "세종"],
      "correct_answer_index": 0,
      "explanation": "조선 제1대 왕은 태조 이성계입니다."
    }
  ]
  ```
- **Local Ranking (LocalStorage)**: `key: 'quiz_ranking'`, `value: Array[{ id, user_name, score, date, category }]`

## 5. 비기능적 요구사항
- **독립형 실행**: 별도의 백엔드 서버 없이 웹 브라우저 단독으로 실행 가능 (Client-side Only)
- **성능**: 도트 애니메이션 및 효과음 재생 시 끊김 없는 60FPS 유지
- **학습 효과**: 오답 시에도 정답 및 짧은 해설 제공
- **반응형 디자인**: 데스크탑뿐만 아니라 모바일 브라우저에서도 픽셀 아트 레이아웃이 깨지지 않도록 최적화

## 6. 기술 스택 (Tech Stack)
- **Framework**: **React** (UI 구성 및 상태 관리)
- **Language**: **JavaScript / TypeScript** (핵심 게임 로직)
- **Styling**: **CSS3** (레트로 스타일 구현, 애니메이션)
- **Asset**: 8-bit 스타일 무료 사운드 효과(SFX) 및 배경음악(BGM), 픽셀 폰트 활용

## 7. 향후 확장성
- 난이도 조절 기능 (쉬움, 보통, 어려움)
- 온라인 멀티플레이어 경쟁 기능 (Firebase 연동)
- 카테고리별 뱃지 및 업적 시스템
